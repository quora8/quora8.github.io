这个隐藏的 Windows 11 技巧将帮助您清理繁忙的桌面

科技
|阅读量:

在 Windows 11 中，微软承诺重新调整操作系统以提高生产力。UI 和用户体验——设计元素、声音、布局和交互——被有意简化。这使 Windows 11 保持井井有条，整洁有序，但其意图在新的多任务处理功能中表现得最为明显。

Windows 11 引入了 Snap Layouts 和 Snap Groups，可让您轻松组织窗口。平铺可以更好地利用屏幕空间，同时保持桌面整洁（通过Microsoft）。然而，尽管有 Snap 功能，但在后台处理多个窗口的多任务最终会变得混乱，因此微软在 Windows 11 中加入了一个漂亮的小手势，以便在发生这种情况时重新吸引你的注意力。轻弹鼠标，您可以隐藏桌面上的所有背景窗口。

该手势称为标题栏窗口抖动。当许多应用程序和 Windows 在后台打开，使您的桌面变得凌乱时，抓住您想要保持活动状态的窗口的标题栏并将其摇动。震动释放了繁忙的桌面，只留下您选择的一个窗口处于活动状态。如果您不熟悉，标题栏指的是每个窗口顶部的功能区，带有最小化、最大化和关闭按钮。Windows 10 默认启用该功能，但您需要在 Windows 11 中找到并打开它。

如何开启摇窗手势
斜齿轮
“设置”应用程序是打开或关闭抖动功能的最简单方法。要启用它：

单击“开始”按钮并输入“设置”，或在“开始”菜单中找到“设置”应用。
向下滚动并选择多任务选项卡。
找到标题栏窗口摇动的按钮。
开机。
您现在可以摇动一个窗口以最小化所有其他窗口。如果“设置”菜单中没有切换开关，这是启用或禁用窗口抖动功能的另一种方法。

斜齿轮
在开始菜单中搜索“本地组策略编辑器”。
打开本地组策略编辑器。组策略编辑器提供管理员访问权限以管理多台计算机（通过Microsoft），但它还列出了操作系统和软件的所有用户首选项。标题栏窗口抖动在其旧 Windows 10 名称 Aero Shake 下列出。
在左侧导航窗格中的用户配置下，展开管理模板文件夹。
选择桌面。
双击标有“关闭 Aero Shake 窗口最小化鼠标手势”的设置。
将打开一个窗口，其中包含三个切换：未配置、启用和禁用。选择“未配置”或“禁用”。
单击应用，然后单击确定。
切换可能看起来违反直觉，但您需要选择已禁用或未配置来激活该功能。启用该设置会关闭摇窗手势。

如何修复 Windows 11 最常见的问题
DIY13/Shutterstock
作者 BHARAT BHUSHAN /更新时间：美国东部时间 2022 年 3 月 7 日上午 10:05
可以说，微软在 Windows 11 上抢了先机，它在发布时就有很多问题和错误。其中一些在随后的更新中得到修复，而另一些即使在今天仍然存在。经过数月的修补，Window 11 不能被称为完美无瑕，因为一些问题仍然存在并且用户不时在论坛上报告。

出现的一些错误是由于更深层次的设置故障而导致的，而一些错误则是由于软件代码中的异常造成的。早些时候，我们缩小了Windows 11 问题及其简单修复的范围。现在，我们挖掘了最近困扰用户的更多常见问题，并提供了详细的解决方案，以使您在 Windows 系统上的整体用户体验更加流畅。

Windows 11 降低 SSD 速度
至关重要的
一些从 Windows 10 升级到 Windows 11 的用户在使用其计算机的固态驱动器 (SSD) 时遇到性能下降的问题。Reddit 社区和微软的支持论坛充满了关于升级影响 SSD 写入速度的投诉。据BetaNews报道，性能下降已通过 CrystalDiskMark 等基准测试得到验证。结果显示，使用相同硬件的 Windows 10，随机写入速度降低了接近 50%。

微软
值得庆幸的是，微软的官方补丁可以轻松解决这个令人头疼的问题：KB5007262 补丁。此更新不会自动推送到所有 Windows 11 系统，因此必须手动安装。任何遇到 SSD 速度问题的人都可以前往Microsoft 更新目录并下载补丁。更新的描述说，更新修补了 NVMe、SSD 和硬盘驱动器性能的问题，这些问题是由于启用了 NTFS USN 日志记录而发生的。显然，磁盘 C: 上的 USN 日志是默认启用的。

升级时出现错误 0x80070070
微软
对于从 Windows 10 升级到新操作系统的用户来说，令人讨厌的错误可能是一个障碍。我们谈论的是错误代码 0x80070070，当没有足够的空间在您的系统上安装所需的设置时会发生这种情况。它主要作为错误代码弹出，字符串为 0x80070070 – 0x50011、0x80070070 – 0x50012 或 0x80070070 – 0x60000。无需删除应用程序和软件来弥补空间，一个简单的修复（通过Windows Central）可以让您的事情变得更轻松。

微软
从系统中删除临时文件将（很可能）提供您继续操作 PC 所需的空间。要从您的 PC 中删除临时文件，请转到“设置”，单击“系统”，然后单击“存储”。在这里，您会找到一个名为“临时文件”的部分。单击它会进一步打开一个部分，您可以在其中选择要删除的临时文件。确保不要删除“Windows 安装程序临时文件”或“Windows ESD 安装文件”，这些是操作系统的升级文件。

谷歌浏览器不工作

如果您无法在 Windows 11 上使用 Chrome，这可能是由多种原因造成的。这可能是由于兼容性问题、文件损坏或应用程序行为冲突所致。要进行故障排除，您可以重新启动系统并重新检查 Chrome 以查看它是否工作正常。否则转到 Google Chrome 图标，右键单击它并选择属性。然后单击兼容性选项卡并选择运行兼容性疑难解答。风险自负，您可能还想尝试禁用您在系统上安装的任何防病毒程序 - 或从其设置中将 Google Chrome 列入白名单。

如果这不能解决问题，您可能需要删除包含 Google Chrome 配置文件数据的文件夹。为此，请按 Windows 键 + R 打开“运行”对话框；在此处输入“%USERPROFILE%\AppData\Local\Google\Chrome\User Data”并跟踪 Default 文件夹并创建它的副本以备份文件。然后删除文件夹。还没有成功？您可能必须通过使用快捷键 Window 键 + X 打开终端来刷新 DNS 缓存（通过Guiding Tech）。在此处键入命令“ipconfig/flushdns”并按 Enter。

如果问题仍然存在，您可能需要通过按 Windows 键 + I 快捷方式来重置网络设置。在窗口中选择“网络和 Internet”，然后单击“高级网络设置”。在“更多设置”选项卡下，选择网络重置选项。

搜索无法正常工作

Windows 11 有一个奇怪的错误，它会冻结搜索栏，导致您无法键入地址或搜索查询。用户还报告说在搜索栏中键入时无法查看文件和应用程序。如果重新启动 PC 或搜索和索引疑难解答等明显的修复程序都没有为您工作，那么第一个选项是手动重新启动 SearchHost.exe。为此，请按 Ctrl+Alt+Del 打开任务管理器并查找“SearchHost.exe”文件。然后选择结束任务并单击结束进程。

或者，您可以通过按 Win+R 打开“运行”对话框来重新启动 Windows 搜索服务（通过Make Use of ）。此后输入“services.msc”以打开服务控制台，并查找 Windows Search。双击它以启动“属性”窗口。现在单击停止按钮，然后单击开始。

对于搜索查询有困难的用户，最好的选择是重建搜索索引。按 Win+I 打开设置，单击隐私和安全，然后在右侧窗格中选择搜索 Windows。在打开的窗口中，选择底部的高级索引选项，然后选择高级按钮。在这里单击重建选项；这将解决索引问题，文件或应用程序现在应该像往常一样显示。

显色错误
一些用户报告了一个非常奇怪的错误（通过Windows 最新），升级的 Windows 11 程序包会错误地在 HDR 屏幕上呈现颜色。对于必须经常使用第三方图像编辑软件的编辑或专业人士来说，这完全是令人沮丧的。当剃掉白色或添加白色或较浅的背景时，这一点最为明显。当它们应该是纯白色时，偏移色调看起来是亮黄色。

微软
微软已经公开承认了这个错误，并详细说明了原因。当 Win32 API 在某些显示器上的特定条件下返回意外信息或错误时，就会发生这种情况。

微软强调该错误不会影响所有颜色配置文件管理程序，并声称他们正在解决影响 Windows 11 版本 21H2 的颜色问题。该修复程序预计将在 1 月下旬推出。遇到此问题的用户可能有一个选择：保持耐心并在 Windows 更新补丁到达时立即安装。微软建议（在上面链接的官方文档中）他们希望在“1 月下旬”之前进行更新和修复。

停止使用谷歌浏览器的 10 个理由

通过 EWDISON 然后/更新：2022 年 2 月 25 日美国东部时间下午 5:15
谷歌或许是一家互联网公司，拥有数十种产品和服务，这些产品和服务围绕着几十年来让我们保持联系的技术。对于大多数人来说，通往互联网的门户实际上是谷歌的另一款产品，Chrome 占据了浏览器市场的最大份额。考虑到它在几乎所有可以想象的计算设备上的使用方式，谷歌浏览器是世界上使用最多的软件，这是有争议的。然而，这并不立即意味着它是体验 Web 的最佳方式，而且如果你能负担得起，你也有一些重要的原因想要避免使用 Chrome。

性能和稳定性
仅仅因为它是最常用的并不意味着它是最好的。有多种因素促成了 Chrome 在市场上的成功。尽管某些功能确实有助于将浏览器推销给用户，但在性能和稳定性方面还远不是最令人印象深刻的。事实上，一些 Chrome 用户甚至可能不情愿地承认他们被迫使用浏览器来实现这个或那个功能，尽管他们对浏览器的性能有很多心痛和头痛。

互联网上充斥着关于Chrome对 RAM 和电池电量的贪得无厌的轶事。在人们越来越依赖硬件资源相对有限的笔记本电脑以及在网络上工作、学习或娱乐的时代，贪婪的网络浏览器可能是他们最不需要的东西。实际上，他们需要做的最后一件事是让 Chrome 崩溃，因为它内存不足，或者更糟糕的是，扩展程序中的一些错误。

公平地说，谷歌一直在努力提高 Chrome 的性能并减少其占用空间，主要是通过限制 Javascript 在后台运行的数量。同时，这有时可能会导致用户体验欠佳，谷歌也在努力避免这种情况。它仍然暗示 Chrome 在默认情况下是一头巨大的野兽，必须驯服并节食。

扩展安全
曾几何时，Web 浏览器在它们支持的第三方附加组件（有时称为扩展）的数量上展开竞争。扩展系统允许浏览器保持苗条，至少与 Internet Explorer 的庞然大物相比，同时为浏览器开发人员未预想甚至不打算实现的功能敞开大门。当然，这要求该软件具有可以连接扩展程序以实现这些功能的挂钩，其中有时包括能够修改用户在网页上看到的内容，甚至可以触摸用户计算机上的文件。

不幸的是，从长远来看，扩展也成为问题的根源，危及浏览器的稳定性以及用户的安全。让事情变得复杂的是谷歌是如何运行其 Chrome 网上应用店的，它甚至比它的 Android 版 Google Play 商店更加开放。作为一个更开放的生态系统的交换，几乎没有任何质量控制，并且许多载有恶意软件的扩展能够从裂缝中溜走，通常伪装成来自信誉良好的开发人员的扩展。

谷歌一直试图填补这个大漏洞，但它的战略是一把双刃剑。它限制了扩展可以访问的内容，以减轻有害扩展的负面影响，但也删除了某些扩展所需的功能。不幸的是，这也不是一个完成的交易，仍然有一些有问题的附加组件通过了谷歌的审查。

设置膨胀
附加组件和扩展本应使浏览器更简单、更轻量级，但最终结果并非总是如此。Chrome 不仅是一款需要大量资源的软件，而且还是一款复杂的软件，其复杂性渗透到其配置选项中。几乎与任何 Google 产品一样，Chrome 具有页面上的设置页面，浏览器本身需要一个迷你搜索引擎来查找适当的控件。

当然，这个“功能”有利有弊。一方面，它让用户可以更好地控制浏览器的行为方式，让他们几乎可以调整体验的各个方面。另一方面，在众多选项中迷失几乎太容易了，而且隐藏重要的隐私或安全设置以阻止用户寻找这些设置也同样容易。确实有一个搜索选项，但前提是您首先知道要查找的内容。

缓慢的功能开发
尽管被认为是“沉重的”，但具有讽刺意味的是，Chrome 在实际添加新功能方面的速度很慢，至少与 Microsoft Edge 甚至 Opera 相比。是的，几乎每个月都会发布一个新的 Chrome 版本（尽管谷歌现在正在调整它），但这个速度只意味着大多数更改都是小规模的，并且专注于修复而不是大的新功能。

这意味着 Chrome 并不总能赶上最新趋势或用户的愿望，但这对某些人来说可能没问题。鉴于浏览器因资源使用而臭名昭著，用户可能肯定更喜欢谷歌专注于润色而不是绒毛。扩展可以添加额外的功能，但谷歌从这些下拉扯地毯的方式可能导致一些心怀不满的开发人员完全离开浏览器。

生态系统锁定
Chrome 是最普遍的互联网网关之一，谷歌的大部分赚钱产品和服务都可以在其中找到。换句话说，它几乎是进入 Google 服务的门户，而且越来越明显的是，Google 正在将其塑造成一个类似的门户。它也可以反过来工作，只有在您已经在使用其他 Google 产品的情况下，才能体验到一些最好的 Chrome 功能。

在过去的几个月里，将其他 Google 产品集成到 Chrome 中的功能越来越多。虽然从商业角度来看这似乎是合乎逻辑的，但在涉及反竞争和垄断商业行为时，它也会引发监管危险信号。它自然会冒着偏爱谷歌其他产品的风险，即使网络浏览器应该对它之外的任何东西都无动于衷。相反，一旦你已经深入了解其他谷歌产品，这也让你更难切换，这要归功于 Chrome。

跟踪和隐私
谷歌从来都不是一个与隐私积极相关的公司，该公司正在努力改变这种形象。在 Web 方面，它一直在大力宣传其努力消除世界上有害的第三方跟踪 cookie，这本身就是一项值得称赞的努力。其他浏览器制造商也加入了这一行列，但并非所有人都同意谷歌最近关于如何做到这一点的建议。

该公司可能主要被视为一家搜索或互联网业务，但谷歌的大部分利润实际上来自其广告业务。互联网广告总是需要某种形式的跟踪，Google 的目标是通过推广“好广告”和“尊重隐私”的跟踪方法让大众更欢迎这些广告和活动。然而，并不是每个人都相信这些承诺，而谷歌在这场战斗中仍然模棱两可。

然而，与此同时，Chrome 正被用作谷歌隐私工作的试验台，特别是旨在阻止恶意广告和跟踪器同时推广好的新技术的新技术。然而，这项工作仍在进行中，仍需通过监管和市场审查，尤其是考虑到它是否真正有利于用户隐私的质疑。

虚假的安全感
即使没有那些即将推出的功能，比如备受争议的FLoC（队列联合学习），Chrome 也确实采取了一些措施来保护用户的隐私。其中一些听起来像您的典型浏览器功能，例如删除站点对某些硬件或隐身模式的访问权限。其他可能特定于 Chrome 或首先在 Chrome 中实现，例如阻止非 HTTPS 站点或删除 FTP 功能。

然而，有时这些功能可能不会像用户认为的那样工作，或者可能有一些警告，谷歌没有很好地解释清楚。最近和最受欢迎的案例之一是 Chrome 的隐身模式，它不像大多数人认为的那样隐身。直到最近，谷歌才真正澄清，在多年来一直提倡使用安全浏览功能后，它确实提供的保护非常少。

隐身模式实际上只会对使用相同设备和相同浏览器或浏览器配置文件的其他人隐藏您的活动。它不会从使用其他方法跟踪您的存在的网站完全隐藏您的活动。它也不总是将您从 Google 隐藏起来，因为 Chrome 仍会出于通常的遥测目的向 Google 的服务器提交一些数据。

谷歌垄断
尽管谷歌承担了从 Chrome 和整个网络中清除有害 cookie 和跟踪器的事业而受到称赞，但人们对该公司别有用心的动机表示怀疑。特别是，它提出的替换这些 cookie 的妥协方案受到质疑，因为它们似乎只对谷歌有利，特别是它的广告平台业务。当然，Chrome 正被用作将其推送到 Web 上的工具。

例如，前面提到的 FLoC 是有争议的，因为它把谷歌和 Chrome 放在了最前沿。例如，它只适用于 Chrome，广告商必须使用谷歌的平台进行审查。它发出一条信息，即在这个新世界中确保网络安全的唯一方法是使用 Chrome 和 Google。自然，其他浏览器制造商以及许多隐私权倡导者和行业监管机构对此并不满意。

浏览器单一文化
即使 Chrome 没有所有这些问题，如果几乎所有平台上的所有人都使用 Google 的浏览器，仍然会出现一个负面影响。缺乏竞争实际上不仅对谷歌而且对整个网络都是危险的，从而导致创新和发展的潜在放缓。

单一文化从来都不是好事，多样性和竞争，至少在可接受的范围内，总是有助于推动创新，即使在软件方面也是如此。其他人可能称其为抄袭，但当竞争对手开始实施新功能时，它们已成为标准，因为它们变得流行或有用。虽然 Chrome 的开源 Chromium 基础已成为许多竞争性网络浏览器的基础，但这些浏览器，如 Microsoft Edge、Opera 和 Brave，正在引入挑战现状的新功能，有时也会改变 Chrome 的发展方向。

决定网络的方向
Chrome 的巨大市场份额的最大问题，以及用户应该转向可能更好的浏览器的原因，可能是它在 Web 的发展方向上给了谷歌太多的影响力。如果 Google 决定 Chrome 将阻止使用这种或那种技术的网站或要求网站实施新功能，大多数网站将不得不进行调整以满足 Google 的需求。有时，这对 Web 有利，例如推动 HTTPS 成为标准。然而，其他时候，谷歌似乎是唯一最终真正受益的人，比如当它“鼓励”网站使用AMP（加速移动页面）时。

然而，网络并不是谷歌的财产，还有很多其他利益相关者，即使他们与 Chrome 和谷歌的市场份额相形见绌。如果其他浏览器制造商和监管机构不保持警惕，不注意地面，谷歌将面临真正的风险和危险，就像谷歌几乎将 FLoC 推到每个人的喉咙里一样。虽然普通用户在这些问题上可能没有太多发言权，但他们仍然可以通过提倡其他浏览器（甚至是基于 Chromium 的浏览器）来阻止谷歌拥有网络，尤其是如果它们提供更好的体验的话。
