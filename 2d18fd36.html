教育如何在元宇宙中发生

近年来，元宇宙的概念获得了极大的普及。

对于许多人来说， Web 3.0提出的理念——去中心化政府、匿名、赚钱以及在与数百万人保持联系的同时获得乐趣——可能太吸引人了，不能简单地忽视。

理解元宇宙可能很困难，但爱好者可能希望尽早参与其中。这不仅适用于他们，也适用于他们的孩子。

深入参与 Web 3.0 的父母很可能会教他们的孩子有关元宇宙本身的知识，但如果这是他们的主要关注点，则很难找到进一步的学习机会。幸运的是，公司正在推动虚拟世界中的教育，允许儿童和成人在基于区块链的虚拟空间中学习。

什么是元宇宙，它如何帮助教育？

对于不熟悉该术语的人，“元界”用于描述虚拟空间的集合。元界寻求将互联网整合到他们自己的沉浸式世界中。

他们部分通过使用加密货币和加密钱包来做到这一点，旨在支持匿名性，同时让用户通过玩游戏和与他人互动来赚钱。许多不同的公司一直在尝试打入元宇宙，其中一些公司创建了自己的。

流行的社交网络公司 Facebook更名为 Meta为了支持他们自己的努力，无数其他科技公司正在寻求与新的元宇宙进行类似的冒险。

对 Web 3.0 的更主流关注为虚拟世界中的教育带来了机会。Edtech，或在教育中使用技术，已经从这些机会中取得了重大进展。Meta 自己在这方面的努力从品牌重塑之初就显而易见，这从他们的概念视频.

在成为 Meta 之前，Facebook 已经非常重视教育。他们提供专注于学习的工具和计划，其中许多今天仍然可以使用。Meta 对 metaverse 教育所做的努力表明，edtech 可以通过几种不同的方式集成到 Web 3.0 中。

其他公司正在通过元宇宙推动教育。有些人喜欢更直接的路线，而另一些人只是希望建立联系，将这些虚拟世界用作广告而不是直接的教育平台。可以通过__ The Sandbox __ 找到一个典型的例子，这是一个寻求将游戏行业与加密货币世界结合起来的大型元宇宙。

沙盒及其教育技术连接

沙盒旨在成为一个非常用户友好的元宇宙。在吹嘘之上自己的知识库，它为玩家提供了简单的方法来访问和创建自己的世界。

当然，随着 The Sandbox 建立在区块链技术之上，它的许多功能旨在奖励或与加密货币一起使用。

这种对如此重要系统的轻松访问保证了相当多的教育，而沙盒已经与莱杰合作利用这一点。

Ledger 在 The Sandbox metaverse 中创建了自己的世界，作为向用户介绍加密安全的一种方式。他们基本上在 The Sandbox 中制作了一款教育游戏，展示了一种可以使用元宇宙进行学习的方式。

其他公司也打算将 The Sandbox 用作元宇宙教育的平台。Chord Hero是一家专注于向他人教授音乐的公司，与沙盒合作将他们的品牌包含在虚拟世界中。

虽然它的努力不如 Ledger 的那么深入，但 Chord Hero 的包容性通过不同的方法推动了教育。通过在 The Sandbox 上做广告，更多的人将了解该公司在向他人传授音乐和演奏乐器方面所做的努力。即使元宇宙不被用作主要的教育媒介，它仍然可以将加密爱好者引导至教育资源。那些投资于 Web 3.0 的人可能会被鼓励将流量引向这些来源，从而允许这种教育间接地由元界驱动。

教育者和学术与元界的整合

与元宇宙的更深入集成为众多教育工作者提供了一个兴趣点。虽然这些集成目前充其量只是表面水平，但主要的学习机构正试图在其环境中利用整个元节概念。

例如，尼科西亚大学有发起了一个开放的元界倡议旨在将大学与 Web 3.0 更完全地结合起来。整个课程和程序都是专门为元节及其功能开发的。该大学甚至将 NFT 画廊作为他们支持区块链技术的一个例子。

其他教育工作者效仿只是时间问题，即使他们的元宇宙概念不包括货币化。

就目前而言，进入元宇宙的最大障碍之一是它与财务的联系。一些教育工作者，尤其是 K-12 项目的教育工作者，因此会犹豫将他们的课程引入元宇宙，这是可以理解的。

然而，元宇宙作为一个概念极大地激发了使用虚拟现实进行学习的灵感。哈佛是一个仔细研究这些概念的地方，它的毕业生和教职员工都在努力观察在元宇宙中学习的未来.

虽然这种教育可能不完全符合元界和 Web 3.0 的概念，但使用“元界”这个术语只是为了进一步联系。即使没有建立在区块链技术之上，虚拟现实教育本质上也将训练人们更容易地与元宇宙集成，让他们能够轻松地学习和使用这些技术。

简而言之，元宇宙中的教育正在快速增长。一些公司选择使用元宇宙作为学习平台，而另一些公司则使用它来重定向到其他教育来源。即使是那些没有完全集成 Web 3.0 的人仍然在提供虚拟世界教育，无论是直接的还是通过多米诺骨牌效应的教与学。元宇宙只会从这里变得更大，任何支持它的人都应该尽早利用这些教育资源。
